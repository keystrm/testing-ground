Arrays in Data Structures and Algorithms

Definition

An array is a collection of elements stored in contiguous memory locations.

Illustration

Index: 0 1 2 3 4
Array: [10, 20, 30, 40, 50]

Key Characteristics

  • Fixed Size: The size of an array is defined at the time of declaration.
  • Index-Based Access: Elements are accessed using their index positions.

Common Operations

1. Access

Time Complexity: O(1)

let arr = [10, 20, 30, 40, 50];
console.log(arr[2]); // 30

2. Insertion

At the End: O(1)

arr.push(60); // [10, 20, 30, 40, 50, 60]

At the Beginning or Middle: O(n)

arr.unshift(0); // [0, 10, 20, 30, 40, 50, 60]
arr.splice(3, 0, 25); // [0, 10, 20, 25, 30, 40, 50, 60]

3. Deletion

From the End: O(1)

arr.pop(); // [0, 10, 20, 25, 30, 40, 50]

From the Beginning or Middle: O(n)

arr.shift(); // [10, 20, 25, 30, 40, 50]
arr.splice(2, 1); // [10, 20, 30, 40, 50]

Practical Uses

  • Storing and accessing a list of items where the size is known and static.
  • Efficiently accessing elements by their index positions.
  • Ideal for implementing other data structures like heaps, stacks, and queues.
