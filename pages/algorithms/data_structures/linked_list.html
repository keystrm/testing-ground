Linked Lists in Data Structures and Algorithms

Definition

A linked list is a collection of nodes where each node contains a value and a pointer to the next node.

Illustration

Head -> [10|*] -> [20|*] -> [30|*] -> [40|*] -> [50|null]

Key Characteristics

  • Dynamic Size: The size of a linked list can grow or shrink dynamically.
  • Node-Based Access: Elements are accessed using nodes that point to the next node.

Common Operations

1. Access

Time Complexity: O(n)

        class ListNode {
            constructor(value) {
                this.value = value;
                this.next = null;
            }
        }

        class LinkedList {
            constructor() {
                this.head = null;
            }

            get(index) {
                let current = this.head;
                for (let i = 0; i < index; i++) {
                    current = current.next;
                }
                return current.value;
            }
        }

        let list = new LinkedList();
        list.head = new ListNode(10);
        list.head.next = new ListNode(20);
        list.head.next.next = new ListNode(30);
        console.log(list.get(2)); // 30

2. Insertion

At the Beginning: O(1)

        list.prepend = function(value) {
            const newNode = new ListNode(value);
            newNode.next = this.head;
            this.head = newNode;
        };
        list.prepend(5); // Head -> [5|*] -> [10|*] -> [20|*] -> [30|*]

At the End: O(n)

        list.append = function(value) {
            const newNode = new ListNode(value);
            if (!this.head) {
                this.head = newNode;
                return;
            }
            let current = this.head;
            while (current.next) {
                current = current.next;
            }
            current.next = newNode;
        };
        list.append(40); // Head -> [5|*] -> [10|*] -> [20|*] -> [30|*] -> [40|null]

3. Deletion

From the Beginning: O(1)

        list.deleteFromBeginning = function() {
            if (!this.head) return;
            this.head = this.head.next;
        };
        list.deleteFromBeginning(); // Head -> [10|*] -> [20|*] -> [30|*] -> [40|null]

From the Middle or End: O(n)

        list.delete = function(value) {
            if (!this.head) return;
            if (this.head.value === value) {
                this.head = this.head.next;
                return;
            }
            let current = this.head;
            while (current.next && current.next.value !== value) {
                current = current.next;
            }
            if (current.next) {
                current.next = current.next.next;
            }
        };
        list.delete(20); // Head -> [10|*] -> [30|*] -> [40|null]

Practical Uses

  • Efficiently adding and removing elements from the beginning of the list.
  • Dynamic data structures where size can change frequently.
  • Implementing other data structures like stacks, queues, and graphs.
