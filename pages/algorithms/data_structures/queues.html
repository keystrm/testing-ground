Queues in Data Structures and Algorithms

Definition

A queue is a linear data structure that follows the FIFO (First In, First Out) principle.

Illustration

Front [10, 20, 30] Rear

Key Characteristics

  • FIFO: First In, First Out.
  • Operations: Enqueue (add), Dequeue (remove), Front (first element).

Common Operations

1. Enqueue

Time Complexity: O(1)

        class Queue {
            constructor() {
                this.items = [];
            }

            enqueue(element) {
                this.items.push(element);
            }
        }

        let queue = new Queue();
        queue.enqueue(10); // [10]
        queue.enqueue(20); // [10, 20]
        queue.enqueue(30); // [10, 20, 30]

2. Dequeue

Time Complexity: O(1)

        Queue.prototype.dequeue = function() {
            return this.items.shift();
        };

        console.log(queue.dequeue()); // 10, Queue: [20, 30]
        console.log(queue.dequeue()); // 20, Queue: [30]

3. Front

Time Complexity: O(1)

        Queue.prototype.front = function() {
            return this.items[0];
        };

        console.log(queue.front()); // 30

Practical Uses

  • Managing tasks in a printer spooler.
  • Handling requests in a web server.
  • Simulation of real-world queues (e.g., ticketing systems).
